РЕГИСТРАЦИЯ ПОЛЬЗОВАТЕЛЯ

Метод: POST /api/users/register

Тело запроса:
  
  {
    "name": "string",
    "email": "string",
    "phone": "string",
    "address": "string",
    "password": "string"
  }
  

Ответ:
  
  {
    "userId": "int",
    "message": "Registration successful"
  }
  

ВХОД В СИСТЕМУ

Метод: POST /api/users/login

Тело запроса:
  
  {
    "email": "string",
    "password": "string"
  }
  

Ответ:
  
  {
    "userId": "int",
    "token": "string",
    "message": "Login successful"
  }

СОЗДАНИЕ ЗАКАЗА

    "userId": "int",

      {
        "productId": "int",
        "quantity": "int"
      }
    ],
    "totalAmount": "float"
  }
  

Ответ:

    "orderId": "int",
    "message": "Order created successfully"
  }
